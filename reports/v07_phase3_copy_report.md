# v07 Phase 3 Copy Smoke and Real Run Report

## Scope

Phase 3 ran the v07 copy task with the frozen Phase 2 canonical graph:

- `N_total=1024`, `copy_source_length=511`, `T=1024`
- `B=32`, `q=32`, `d=8`, causal attention
- canonical graph sha256: `53ae37a6584833a1d20d51162a01a03d18bf7eeb9fd0efd2ebf3b8a482427a48`
- remote conda env: `ysx_base`, GPU class: `NVIDIA A100-SXM4-80GB`

The first smoke attempt failed because the remote environment lacks `matplotlib`. The partial failure output was preserved remotely, and code commit `a143e01` added a no-matplotlib PNG fallback. The successful smoke and real runs below used commit `a143e01` to keep copy Phase 3 code provenance stable.

  2>&1 | tee logs/copy_v07_smoke_n1024_q32_B32_d8_20260614T112121Z.log

Real run started on GPU3 for `dense`, `local`, and `zigzag_certified`. After the user approved using all four GPUs, the remaining three methods were launched in independent method-only runs:

- GPU2: `zigzag_certified_cosine`
- GPU1: `random_regular`
- GPU0: `zigzag_boolean`

The duplicate sequential `zigzag_certified_cosine` that began after `zigzag_certified` finished was interrupted and preserved under `outputs/copy_v07_main_n1024_q32_B32_d8/interrupted_duplicate/`. The canonical main output was then merged from the six completed method runs.
